EVENT-US 개요

누구나 클릭 몇 번이면 행사에 필요한 기능들과 디자인을 갖춘 전문 플랫폼을 만들 수 있는 행사전용 클라우드 서비스입니다.

  • CLOUD SERVICE

    클라우드 서비스로써 행사전용 플랫폼을 제공하여 지속적으로 사용가능

  • POWER SOLUTION

    행사에 필요한 대부분 프로세스들을 대체해줄 ICT 솔루션들을 제공

  • MODERN DESIGN

    Google Material Design을 기반으로 깔끔하고 간편한 UX/UI 제공

  • EASY CUSTOMIZE

    누구나 쉽게 행사에 최적화되게 기능 설정이나 디자인,변경이 가능

행사 참가자들이 EVENT-US를 이용하는 방법

스마트폰, 태블릿, PC 어디서든 인터넷 브라우저에 접속할 수 있으면 이용 가능합니다.

01

Web 플랫폼 접속

행사 주최측에서 안내해준 전용주소를 인터넷 주소창에 입력

02

행사 참가자 구분

플랫폼에 안내된 대로 닉네임 입력 또는 신청하실 때 입력하셨던 핸드폰 번호 또는 이메일 주소를 입력

03

기능 선택하여 참여

행사의 진행상황에 따라 필요하신 기능을 클릭하여 행사에 적극적으로 참여
